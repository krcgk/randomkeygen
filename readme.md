# Random Keygen

A cli tool for quick random password and secret key generator

### Usage 

```bash
npm i -g randomkeygen

randomkeygen
# JM7jfXJOo9bt

randomkeygen strong
# Zlre6Ln#?zq:N;G&vDOyS8xff5ht=.?EQK

randomkeygen ultra
# @*@~0>vc$epmNDaJxAEif[<X7!oCVwATk[$TPXx;W=fCNWhT?z

randomkeygen -l 16 -c numeric
# 1611056464649654
```
